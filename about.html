About

FleakOS is a system that was born with student student enthusiasts who resolved their skills in practice by creating a unix-like system in a way that was able to maintain a system for different people. From an ordinary user to a more experienced developer. The premise is, if you have a light and safe system, almost without any failure (although this is almost impossible).

We thought about having a system without any changes that would add the term "bloated", we do not want to have a poorly done system when the use is exactly the experience and the security within it. In other words, we believe that more is less. The system, therefore, can leave the user more free to customize and configure it the way they want.
