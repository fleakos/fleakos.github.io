Getting Started

This is the download page, scroll down a bit more, and you can find complete installation documentation, a table with the main architectures, and that, if yours is not listed, it may be in the complete list. We still provide some quick setup tutorials.

Remember to always check your architecture, also remember to have the necessary knowledge to generate partitions, since the system does not offer you any program or script that creates an assisted partitioning.
Note: For a more comfortable use of the system, we recommend more than 500M of memory in the / boot / efi partition.

CPU Machine Media Installation
amd64 64-bit x86-family machines with AMD or Intel CPUs USB image
i386 32-bit x86-family generic machines USB image, CD
arm ARM systems like Raspberry Pi, PINE64, ODROID, "ServerReady" USB image
mips MIPS systems like EdgeRouter, Loongson, Malta USB image, CD

The full list of the CPUs

Note: If your architecture does not even appear in the table, let alone the full list of supported architectures, unfortunately it has not yet been supported. However, if you wish, send us an email contacting us about it.

You can still check all live images and tarball files in the official repository. You can download your image on the Mirror of your choice, however, it is recommended that you use a local repository (as close to you as possible).

https://repo.official.flakeos.org/archives/current

You can also check the integrity of the files with the following command below:

# sha256 youriso.iso
